JuliusJS是一个语言识别的 JavaScript 库。可以先试试这个在线示例，对着话筒说英文测试。

// bootstrap JuliusJS
var julius = new Julius();

julius.onrecognition = function(sentence) {
    console.log(sentence);
};

// say "Hello, world!"
// console logs: `> HELLO WORLD`

特性

  • 实时转录
  • 利用现有的语法，也可自定义
  • 100% JavaScript 实现
  • 在浏览器中通过 Worker 完成识别操作
  • 熟悉的基于事件的 API
  • 没有额外的服务器调用
